Reconstrucing charged particle track segments with a quantum-enhanced support vector machine

We calculate what is called the 'kernel matrix' on a quantum device and use that to define a machine learning model (a support vector machine) which is used to classify small connections between hits left by particles in a detector as belonging (or not) to a particle track.
